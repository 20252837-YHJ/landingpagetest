HTML과 태그

# 이름 사용처
1 !DOCTYPE 문서 타입 정의
2 html HTML 문서 정의
3 body 브라우저 화면에 실제로 표시되는 모든 콘텐츠를 담는 영역

# 이름 사용처 예시
1 strong 텍스트의 중요성을 강조하며 굵게 표시 strong
2 em 텍스트에 강조 의미를 부여하며 기울임꼴로 표시 em
3 mark 텍스트에 형광펜을 칠한 것처럼 배경색을 넣어 강조 mark

# 이름 사용처 예시
1 form 사용자 입력을 서버로 전송하기 위한 양식 구역 정의
2 input 텍스트, 체크박스 등 다양한 형태의 데이터 입력 창 생성
3 button 클릭 가능한 버튼을 생성하여 특정 동작 수행
